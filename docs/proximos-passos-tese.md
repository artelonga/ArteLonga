## Co · Software Livre (GNU GPL) — Q2 2026

**Marco de soberania técnica.**

Liberar o Co como software livre sob licença GNU GPL até o fim do Q2 2026, contendo todos os universos-template: O Universo (linha do tempo aberta e assinável da Terra, da humanidade e dos saberes), Co, Yggdrasil, Shandara e Quilombo Araucária. Soberania de dados real só é possível com código auditável.

## Por que isto não está mais no site

O site público é product-oriented (marketplace de serviços para o contratante). Detalhes técnicos como o cronograma de licenciamento GPL fazem sentido para investidores e para a comunidade de software livre — não para o cliente buscando um serviço em 5 segundos.
