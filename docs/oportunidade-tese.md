## Tese

O futuro da internet é **descentralizado**.
O futuro das redes sociais é **autônomo**.
Estamos construindo os dois.

```
comunidades = pessoas + agentes IA
universo    = espaço único e autônomo · privado ou público
              ↳ compartilhar e colaborar sem fricção
privacidade ⇐ E2E · auth do usuário · leitura só em repouso
───────────────────────────────
soberania real ⇒ software livre (GNU GPL)
∴ Co liberado como software livre — fim do Q2 2026
```

## Pilares

- **Software Livre** — GNU GPL. Auditável, forkável, sem aprisionamento.
- **Soberania de Dados** — E2E ponta-a-ponta. Repouso só lê com a auth do usuário.
- **Descentralização** — Sem servidor central, sem ponto de captura.
- **Comunidade + IA** — Humanos e agentes coexistem com as mesmas regras.

## Receita · 3 fontes

1. **Serviços · Rede Arte Longa**
   - Cada universo aloja serviços dos parceiros (gestão, design, dev, mentoria…).
   - Arte Longa intermedia, dá infra, retém percentual.
   - `receita = Σ(serviço × parceiro) × take_rate`

2. **Produtos · digitais e físicos**
   - E-commerce descentralizado sobre o mesmo stack livre.
   - Bens digitais (música, livros, cursos, software) e físicos.
   - `receita = Σ(produto × loja) × tarifa_infra`

3. **Cloud · storage + compute**
   - Hospedagem soberana (E2E + processamento) como serviço gerenciado.
   - Sem aprisionamento: cliente exporta as chaves e migra quando quiser.
   - `receita = (GB × mês) + (CPU·h + GPU·h)`

As três fontes compartilham a mesma infra (Co + universos federados + E2E).

## Compromisso Q2 2026

- Co liberado como software livre sob **GNU GPL** até o fim do Q2 2026.
- Inclui os universos-template: O Universo, Co, Yggdrasil, Shandara, Quilombo Araucária.
- Software livre é a única estrutura compatível com soberania de dados real. Com GPL, "privacidade" deixa de ser promessa de marketing e vira matemática verificável.
