Design Palette

Sistema de design do site Arte Longa em isolamento. Mostra form: cores, tipografia, componentes, padrões. Content (parceiros, serviços específicos) vive em outro lugar — <handle>/profile.yaml, data.js, etc. Aqui é só template visual. Open this page when you need to see how a component looks before composing pages.

How to use: cada component tem um canvas (visual) + code (markup) + uma API table com a interface TS real. Os canvases marcados com data-showcase são populados pelo módulo real em /dist/showcase.js — não HTML mock.

Color palette

Monochromatic com 1 accent (vermelho terra). Background sempre branco; tipografia preta; greys pra hierarquia secundária.

Buttons

Padronização: primário sólido preto sobre branco, secondário borda só. Hover: invertem (background→branco→preto, etc).
