GIVEN handles são referenciados de várias seções (`services.responsavel[]`,
`citacoes.autor`, `community.membros[]`, `profile.subMembers[]`,
`portfolio.autorEmBreve` etc), AND um handle citado sem entrada
correspondente em `AL.people` ou `AL.communities` quebra silenciosamente
em runtime (renderer pula ou crash), AND nenhum audit captura isso hoje
(audit-consistency só checa profile↔community direção),

WHEN adicionamos `tools/audit-handles.mjs` que itera todas as
references de handle em data.js (services, citacoes, community.membros,
parentHandle, subMembers, contribuicoes.quem, etc), valida que cada
`handle` resolve via `AL.get(handle)`,

THEN edição que introduz typo em handle (`yur` em vez de `yuri`) ou
remove perfil sem limpar referências falha em audit time, antes de
chegar em prod.

## Implementação

```js
// tools/audit-handles.mjs
const ALL_HANDLES = new Set([
    ...AL.people.map(p => p.handle),
    ...AL.communities.map(c => c.handle),
]);

const refs = [];
// services.responsavel + services.implicitResponsavel
for (const s of AL.services) {
    for (const h of (s.responsavel || [])) refs.push({ src: `service "${s.titulo}".responsavel`, handle: h });
    for (const h of (s.implicitResponsavel || [])) refs.push({ src: `service "${s.titulo}".implicitResponsavel`, handle: h });
}
// citacoes.autor (people)
for (const p of AL.people) {
    for (const c of (p.citacoes || [])) {
        if (c.autor) refs.push({ src: `${p.handle}.citacoes[].autor`, handle: c.autor });
    }
    for (const h of (p.subMembers || [])) refs.push({ src: `${p.handle}.subMembers`, handle: h });
    if (p.parentHandle) refs.push({ src: `${p.handle}.parentHandle`, handle: p.parentHandle });
    for (const h of (p.communities || [])) refs.push({ src: `${p.handle}.communities`, handle: h });
}
// community.membros + parcerias.contribuicoes
for (const c of AL.communities) {
    for (const h of (c.membros || [])) refs.push({ src: `${c.handle}.membros`, handle: h });
    for (const pa of (c.parcerias || [])) {
        if (pa.de) refs.push({ src: `${c.handle}.parcerias.de`, handle: pa.de });
        for (const co of (pa.contribuicoes || [])) refs.push({ src: `${c.handle}.parcerias.contribuicoes.quem`, handle: co.quem });
    }
}

const orphans = refs.filter(r => !ALL_HANDLES.has(r.handle));
if (orphans.length) { /* exit 1, list */ }
```

## Out of scope

- Validation de tipo (handle existe MAS é wrong type — ex citação aponta pra community). Future audit se necessário.
- Reverse audit (perfil sem nenhuma referência = órfão da rede). Diferente concern.
