## As

a visitor on artelonga.com.br who wants to create an account (to join
universes, save preferences, send DMs, get notified) without leaving
the artelonga.com.br domain

## I Need

a signup form on artelonga.com.br that takes my email, sends me a
6-digit code via the CO backend, accepts the code, and lands me with
a session that works across the `.artelonga.com.br` subdomain
(including co.artelonga.com.br, quilomboaraucaria.org via SSO bridge)

## So That

artelonga.com.br is no longer a static dead-end for new users — they
can sign up here and be onboarded into the CO ecosystem with origin
"artelonga" tagged, useful for future analytics and campaign tracking.

---

## Backend dependency

**CO-205 ships the backend** (CORS for artelonga.com.br + origin

## What this ticket adds

### 1. New `/entrar/` page

```
artelonga.com.br/entrar/
├── index.html       — the form + code-entry view
└── ../assets/al-signup.js — the JS module (or co-located)
```

Two-step layout:

```
┌──────────────────────────────────────┐
│ Entrar / Cadastrar                  │
│                                      │
│ Continue com seu email:              │
│ ┌──────────────────────────────────┐ │
│ │ email@exemplo.com                │ │
│ └──────────────────────────────────┘ │
│            [ Continuar ]             │
│                                      │
│ Ou com Google:                       │
│            [ G Google ]              │
└──────────────────────────────────────┘

After submit → flips to:

┌──────────────────────────────────────┐
│ Verifique seu email                  │
│                                      │
│ Enviamos um código para              │
│ guest@example.com                    │
│                                      │
│ ┌────────────┐                       │
│ │ 6 dígitos  │                       │
│ └────────────┘                       │
│            [ Confirmar ]             │
│                                      │
│ Não recebeu? [Reenviar] · [Editar]   │
└──────────────────────────────────────┘
```

After successful verify → redirect to `/` with logged-in state OR
to `co.artelonga.com.br` (the universe board). Default: stay on
artelonga.com.br and reflect logged-in state in the header.

### 2. JS module — `assets/al-signup.js`

```js
const CO_BASE = 'https://co.artelonga.com.br';

const form = document.getElementById('al-signup');
const codeStep = document.getElementById('al-code-step');
const emailStep = document.getElementById('al-email-step');
const errEl = document.getElementById('al-signup-error');
const resendBtn = document.getElementById('al-resend');
let _email = '';
let _resendCooldown = null;

form?.addEventListener('submit', async (e) => {
    e.preventDefault();
    errEl.hidden = true;
    const email = form.email.value.trim();
    if (!email.includes('@')) {
        errEl.textContent = 'Email inválido';
        errEl.hidden = false;
        return;
    }
    const btn = form.querySelector('button[type=submit]');
    btn.disabled = true;
    btn.textContent = 'Enviando…';
    try {
        const r = await fetch(`${CO_BASE}/api/v1/auth/onboard-with-email`, {
            method: 'POST',
            credentials: 'include',
            headers: { 'Content-Type': 'application/json' },
            body: JSON.stringify({ email, origin: 'artelonga' }),
        });
        if (!r.ok) throw new Error('send failed');
        _email = email;
        emailStep.hidden = true;
        codeStep.hidden = false;
        document.getElementById('al-code-target').textContent = email;
        document.getElementById('al-code').focus();
        startResendCooldown();
    } catch (e) {
        errEl.textContent = 'Não foi possível enviar. Tente novamente.';
        errEl.hidden = false;
    } finally {
        btn.disabled = false;
        btn.textContent = 'Continuar';
    }
});

document.getElementById('al-verify-form')?.addEventListener('submit', async (e) => {
    e.preventDefault();
    const code = document.getElementById('al-code').value.trim();
    if (code.length !== 6) return;
    const btn = document.getElementById('al-verify-btn');
    btn.disabled = true;
    btn.textContent = 'Verificando…';
    try {
        const r = await fetch(`${CO_BASE}/api/v1/auth/onboard-with-email/verify`, {
            method: 'POST',
            credentials: 'include',
            headers: { 'Content-Type': 'application/json' },
            body: JSON.stringify({ email: _email, code }),
        });
        if (!r.ok) throw new Error('verify failed');
        // Cookie now set on .artelonga.com.br — page can detect via /auth/me
        window.location.href = '/';
    } catch (e) {
        const errEl2 = document.getElementById('al-verify-error');
        errEl2.textContent = 'Código inválido ou expirado.';
        errEl2.hidden = false;
    } finally {
        btn.disabled = false;
        btn.textContent = 'Confirmar';
    }
});

function startResendCooldown() {
    let secs = 60;
    resendBtn.disabled = true;
    resendBtn.textContent = `Reenviar (${secs}s)`;
    _resendCooldown = setInterval(() => {
        secs -= 1;
        if (secs <= 0) {
            clearInterval(_resendCooldown);
            resendBtn.disabled = false;
            resendBtn.textContent = 'Reenviar';
        } else {
            resendBtn.textContent = `Reenviar (${secs}s)`;
        }
    }, 1000);
}

document.getElementById('al-edit-email')?.addEventListener('click', () => {
    codeStep.hidden = true;
    emailStep.hidden = false;
});

resendBtn?.addEventListener('click', () => {
    if (resendBtn.disabled) return;
    // Re-submit the email step
    form.dispatchEvent(new Event('submit'));
});

// Google sign-in button
document.getElementById('al-google-btn')?.addEventListener('click', () => {
    const returnTo = encodeURIComponent(window.location.origin + '/');
    window.location.href =
        `${CO_BASE}/api/v1/auth/google/start?origin=artelonga&return_to=${returnTo}`;
});

// On page load, check if already signed in
(async () => {
    try {
        const r = await fetch(`${CO_BASE}/api/v1/auth/me`, { credentials: 'include' });
        if (r.ok) {
            // Already logged in — redirect to homepage
            window.location.href = '/';
        }
    } catch (_) {}
})();
```

### 3. Homepage CTA

Add a small "Entrar / Cadastrar" link in the main nav of `/index.html`
that goes to `/entrar/`. Optional inline mini-form on the homepage
that does the email step (then redirects to /entrar/ for code entry).

### 4. "Logged in" indicator in the header

When `GET /api/v1/auth/me` returns 200 with a user, the header shows:

```
Olá, {display_name}  ·  [ Sair ]
```

Logout calls `POST /api/v1/auth/logout` (cookie cleared on
`.artelonga.com.br`) then reloads.

### 5. Style — match existing artelonga aesthetic

Reuse `.fp-form` patterns from `/faca-parte/` (large uppercase
headings, generous spacing, dark on light). Don't introduce new
design tokens.

## Acceptance

- [ ] New `/entrar/` page renders the two-step signup form
- [ ] Email step POSTs to CO onboard endpoint, transitions to code step on 202
- [ ] Code step POSTs verify, on success sets cookie and redirects to `/`
- [ ] Invalid email shows inline error without server roundtrip
- [ ] Invalid code shows inline error from server response
- [ ] Resend cooldown 60s after each send
- [ ] "Editar email" returns to step 1
- [ ] Google button hits CO's Google OAuth start with `origin=artelonga`
- [ ] On page load, if already logged in (via `/auth/me`), redirect to `/`
- [ ] Header on `/index.html` shows logged-in state when session exists
- [ ] Logout button clears cookie and refreshes
- [ ] Mobile responsive (single column, full-width form on <768px)
- [ ] All strings in pt-BR matching existing site voice
