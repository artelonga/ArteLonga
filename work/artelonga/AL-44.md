GIVEN audit pipeline tem audit-shells + audit-consistency + validate-yaml,
mas não tem audit pra handles órfãos em cross-references, AND STATE.md
roda manual via `bake-state.mjs` (fica stale), AND ambos são adições
small de tooling complementar,

WHEN adicionamos `tools/audit-handles.mjs` (cross-reference check) +
`.github/workflows/bake-state.yml` (mensal auto-regen de STATE.md),

THEN edição que introduz typo em handle (`yur` vs `yuri`) ou remove
perfil sem limpar referências falha em audit time; STATE.md fica

## Parte 1: `tools/audit-handles.mjs`

Itera todas as references de handle em data.js (services, citacoes,
community.membros, parentHandle, subMembers, contribuicoes.quem),
valida que cada `handle` resolve via `AL.get(handle)`.

```js
const ALL_HANDLES = new Set([
    ...AL.people.map(p => p.handle),
    ...AL.communities.map(c => c.handle),
]);

const refs = [];
// services.responsavel / implicitResponsavel
for (const s of AL.services) {
    for (const h of (s.responsavel || [])) refs.push({ src: `service "${s.titulo}".responsavel`, handle: h });
    for (const h of (s.implicitResponsavel || [])) refs.push({ src: `service "${s.titulo}".implicitResponsavel`, handle: h });
}
// people: citacoes.autor, subMembers, parentHandle, communities
for (const p of AL.people) {
    for (const c of (p.citacoes || [])) {
        if (c.autor) refs.push({ src: `${p.handle}.citacoes[].autor`, handle: c.autor });
    }
    for (const h of (p.subMembers || [])) refs.push({ src: `${p.handle}.subMembers`, handle: h });
    if (p.parentHandle) refs.push({ src: `${p.handle}.parentHandle`, handle: p.parentHandle });
    for (const h of (p.communities || [])) refs.push({ src: `${p.handle}.communities`, handle: h });
}
// community.membros + parcerias.de + parcerias.contribuicoes.quem
for (const c of AL.communities) {
    for (const h of (c.membros || [])) refs.push({ src: `${c.handle}.membros`, handle: h });
    for (const pa of (c.parcerias || [])) {
        if (pa.de) refs.push({ src: `${c.handle}.parcerias.de`, handle: pa.de });
        for (const co of (pa.contribuicoes || [])) refs.push({ src: `${c.handle}.parcerias.contribuicoes.quem`, handle: co.quem });
    }
}

const orphans = refs.filter(r => !ALL_HANDLES.has(r.handle));
if (orphans.length) { /* exit 1 + list */ }
```

Adicionar `npm run audit-handles` em package.json. Estender combo
`npm run audit` pra incluir audit-handles.

## Parte 2: `.github/workflows/bake-state.yml`

```yaml
name: bake-state
on:
  schedule: [{ cron: '0 3 1 * *' }]
  workflow_dispatch: {}
permissions: { contents: write }
jobs:
  bake:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4
        with: { node-version: '20' }
      - run: npm ci
      - run: node tools/bake-state.mjs
      - uses: stefanzweifel/git-auto-commit-action@v5
        with:
          commit_message: "chore(state): auto-bake STATE.md"
          file_pattern: docs/STATE.md
```
