GIVEN site não tem testes automatizados — regressões só pegas em
smoke test manual pós-merge, AND não há audit a11y formal (cores
cinza claro em fundo branco, aria-labels possivelmente faltando,
contrast não medido), AND não há perf budget (bundle renderer.js 77KB,
LCP/CLS não medidos),

WHEN configuramos uma única CI quality pipeline rodando em PRs com
Playwright (smoke + a11y via axe) + Lighthouse CI (perf + SEO), AND
definimos budget que bloqueia merge se quebrar,

THEN PR que regredir UX, a11y ou performance é caught antes de prod.
Uma única feature unifica 3 concerns relacionados (todos = quality
gates em PR).

---

## Stack escolhida

- **Playwright** — smoke + a11y. Único runner, single config.
- **@axe-core/playwright** — a11y rules embeddadas nos smoke tests.
- **Lighthouse CI (@lhci/cli)** — perf + SEO + best practices.

## Parte 1: Playwright smoke + a11y

`tests/e2e/smoke.spec.ts`:

```ts
import { test, expect } from '@playwright/test';
import AxeBuilder from '@axe-core/playwright';

const BASE = process.env.BASE_URL || 'http://localhost:8000';

const pages = [
    { path: '/', h1: /Serviços|Marketplace/ },
    { path: '/parceiros/', h1: /Parceiros/ },
    { path: '/yuri/', h1: /Yuri/ },
    { path: '/yuri/maes/', h1: /Mães/ },
    { path: '/kiyoshi/inocencia/', h1: /Inocência/ },
    { path: '/quilomboaraucaria/', h1: /Quilombo/ },
    { path: '/servicos/grafite/', h1: /Grafite/ },
    { path: '/contato/', h1: /Contato/ },
    { path: '/design/', h1: /Design/ },
];

for (const p of pages) {
    test(`smoke + a11y: ${p.path}`, async ({ page }) => {
        const errors: string[] = [];
        page.on('console', m => { if (m.type() === 'error') errors.push(m.text()); });

        const resp = await page.goto(BASE + p.path);
        expect(resp?.status()).toBe(200);

        // L-002: no fallback visible
        await expect(page.locator('text=Algo quebrou')).toHaveCount(0);

        // No JS errors
        expect(errors).toHaveLength(0);

        // Page-specific
        await expect(page.locator('h1')).toContainText(p.h1);

        // A11y (only critical/serious; warns documented)
        const results = await new AxeBuilder({ page })
            .withTags(['wcag2a', 'wcag2aa'])
            .analyze();
        const blockers = results.violations.filter(v => v.impact === 'critical' || v.impact === 'serious');
        expect(blockers, JSON.stringify(blockers.map(b => b.id), null, 2)).toHaveLength(0);
    });
}
```

## Parte 2: Lighthouse CI

`.lighthouserc.js`:

```js
module.exports = {
    ci: {
        collect: {
            url: [
                'http://localhost:8000/',
                'http://localhost:8000/parceiros/',
                'http://localhost:8000/yuri/',
                'http://localhost:8000/servicos/grafite/',
                'http://localhost:8000/contato/',
            ],
            numberOfRuns: 3,
        },
        assert: {
            assertions: {
                'categories:performance': ['error', { minScore: 0.9 }],
                'categories:accessibility': ['error', { minScore: 0.9 }],
                'categories:seo': ['error', { minScore: 0.95 }],
                'categories:best-practices': ['warn', { minScore: 0.9 }],
                'largest-contentful-paint': ['error', { maxNumericValue: 2000 }],
                'cumulative-layout-shift': ['error', { maxNumericValue: 0.1 }],
                'total-byte-weight': ['warn', { maxNumericValue: 500000 }],
            },
        },
        upload: { target: 'temporary-public-storage' },
    },
};
```

## Parte 3: GH Action `.github/workflows/quality.yml`

```yaml
name: quality
on: [pull_request, push]
jobs:
  quality:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4
        with: { node-version: '20' }
      - run: npm ci
      - run: npm run build
      - run: npx playwright install --with-deps chromium
      - run: npx serve . -p 8000 &
      - run: sleep 2
      - name: Playwright smoke + a11y
        run: BASE_URL=http://localhost:8000 npx playwright test
      - name: Lighthouse CI
        run: npx @lhci/cli@latest autorun
      - uses: actions/upload-artifact@v4
        if: failure()
        with: { name: quality-artifacts, path: test-results/ }
```
